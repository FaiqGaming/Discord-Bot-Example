# How to run the bot

You need [Node.js](https://nodejs.org/) version 8.x (8+) to run the bot.

Head to the bot directory and then simply run :

## Executing

### From the terminal
```sh
$ npm i -S
$ node index.js
```

### From the bat file

Just run the `runbot.bat`
